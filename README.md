## Temas Vistos en Clase

### Introducción

En esta primera unidad aprendimos los fundamentos de como funciona una computadora, desde su base fisica (hardware) hasta los conceptos iniciales de programación.
Hardware y representación simbólica el hardware es la parte física del computador: los componentes que podemos tocar, como la memoria, el procesador o las tarjetas de almacenamiento (por ejemplo una SSD)
Atraves del ejemplo del ábaco, entendimos cómo los números - como 25 y 30- pueden representarsen de 2 maneras: 
Simbolica: el numero que escribimos y entendemos los humanos.
Física: la forma en que la máquina lo representa internamente (por ejemplo, corriente eléctrica o cargas en un chip).
Así comprendimos que, para la computadora, todo se traduce en señales que represenran ceros y unos, lo que nos lleva al siguiente punto.

### Sistema Binario

Sistema binario vinmos que las computadoras trabajan en base 2 donde solo existen dos estados posibles:
0-> sin corriente falso 
1-> con corriente verdadero 
Con esta base se pueden representar todos los números y operaciones.
por ejemplo:
25 en binario es 11001 
30 en binario es 11110
su ,suma es 55, se representa como 110111

### Logica booleana y circuitos 

Conocimos a George Boole (1846), quie desarrolló las bases de la lógica matematica, y a Claude Shannon (1937), quien aplicó esa lógica al diseño de circuitos eléctricos.
Gracias a ellos, hoy existen los circuitos lógicos que permiten hacer operaciones dentro de los procesadores.


Aprendimos las operaciones lógicas básicas:
AND -> verdadero solo si ambas entradas son 1 
OR -> verdadero si almenos una es 1 
NOT -> invierte el valor (de 1 a 0 o de 0 a 1 )
XOR -> verdadero si las entradas son diferentes 
Estas operaciones se combinan para formar un sumador lógico (ADDER),prite a la computadora realizar operaciones aritméticas.


### ¿Que es un programa?

Un programa es un conjunto de instrucciones que la computadora ejecuta para resolver un problema o realizar una tarea .
Estas instrucciones estan escritas en un lenguaje de programación, que traduce nuestras ideas (nivel simbólico) a un nivel que la maquina puede entender (binario).


### ¿Que son variables?

Las variables son espacios de memorias donde guardamos información temporalmente.
Cada variable tiene un nombre y un valor, y nos permite manipular datos dentro de un progama.
por ejemplo, si definimos una variable edad =20, estamos almacenando el numero 20 en un espacio de memoria identificado como "edad".

### Ejemplo básico de programa en Python
```python
# Definimos variables
a = 25
b = 30

# Sumamos las variables
suma = a + b

# Mostramos resultados
print(f"El resultado de sumar {a} y {b} es {suma}")
```
